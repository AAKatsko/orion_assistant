# Synapse – Ассистент для поддержки сотрудников и клиентов компании

## Кейс
Разработка AI-ассистента для мгновенного поиска информации в корпоративной технической документации с целью ускорения процессов поддержки сотрудников и клиентов.

## Проблема
Сотрудники (инженеры, BDM, presale) тратят много времени на поиск информации в разрозненной технической документации. Для ответа на вопросы клиентов или партнёров часто требуется ожидание ответа от узкопрофильного специалиста. Это замедляет циклы продаж, подготовку ценностных предложений и снижает общую эффективность работы.

## Цель
Создать инструмент, работающий как внутренний «умный» ассистент:
- Сотрудник задаёт вопрос на естественном языке.
- Мгновенно получает точный ответ с указанием источника (документа).
- Ответы должны быть верифицированы через цитирование исходных документов.

## Функциональность
- **Автоматическая индексация и векторизация** корпоративной документации (PDF), хранящейся на Яндекс.Диске.
- **RAG-пайплайн** на базе LLM Qwen2.5-32B.
- **Цитирование источников** с указанием на имя документа и страницу.
- **Web-интерфейс** в стиле чат-бота.

## Сценарий использования
1. Пользователь открывает web-интерфейс **OrionGPT**.
2. Вводит вопрос, например:  
   *«Какие продукты Orion soft интегрируются с ОС Astra Linux и какие есть ограничения?»*
3. Система:
   - Автоматически анализирует индексированные документы.
   - Формирует ответ на основе релевантных фрагментов.
   - Возвращает:
     - Прямой ответ, сгенерированный LLM.
     - Список документов-источников с указанием страниц и цитатами.
